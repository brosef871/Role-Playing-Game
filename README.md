# Role-Playing-Game

#	This Role-Playing-Game is an experimental project to practice constructing systems and other applications of software management.
	Objectives:
		1.	Clearly labeling and documenting proper designs for all to follow along, and/or construct themselves for their own purpose.
		2.	Follow engineering ethics and aim to bring new perspective to my software abilities
		3.	Create a fun and enjoyable game. Endless and Evolveable.
